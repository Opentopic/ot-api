Installation
============

.. note:: Client was tested with Python 3.5.1

There are two ways of how you can install Opentopic Api Client.

1. Installing from PyPI - the Python Package Index (last official version)

::

    pip install opentopic-api-client


2. Installing from github repository - http://github.com/opentopic/ot-api/

::

    pip install -e git+https://github.com/opentopic/ot-api.git#egg=ot-api


.. note:: To use Api you need to have your Opentopic account slug, username and password.
